# walkonline.site

A high-resolution, 3D-themed landing page for **walkonline.site** — AI-powered web development.

## 🚀 Hosting on GitHub Pages

1. Create a new GitHub repository (e.g. `walkonline-site`).
2. Upload **all** the files in this folder (`index.html`, `styles.css`, `assets/` folder, `README.md`).
3. Go to **Settings → Pages**.
4. Under "Source", select branch **main** and folder **/ (root)**.
5. Click **Save**. Your site will be live at `https://<username>.github.io/<repo-name>/` in ~1 minute.

### Custom domain
To use `walkonline.site`:
1. In **Settings → Pages → Custom domain**, enter `walkonline.site`.
2. At your domain registrar, add a CNAME record pointing to `<username>.github.io`.

## 📁 Files
- `index.html` — main landing page
- `styles.css` — all styling
- `assets/` — logo + 3D robot images

## ✏️ Customize
- Update WhatsApp number in `index.html` (search for `919999999999`)
- Update email (`hello@walkonline.site`)
- Replace images in `assets/` with your own if desired
